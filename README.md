# arduino-climate-control
## Components
- Arduino Uno
- TMP36 Temperature Sensor
- TIP120 Transistor
- DC Motor
- 220Ω Resistor

## Description
This project controls a DC motor (fan) based on temperature.

- Below 27°C → Fan OFF
- 27°C–40°C → Fan speed increases gradually
- 40°C and above → Fan runs at full speed
